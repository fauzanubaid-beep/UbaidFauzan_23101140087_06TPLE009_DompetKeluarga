# Dompet Keluarga 💳

**Dompet Keluarga** adalah sebuah aplikasi Android pintar untuk mencatat, merencanakan, dan mengelola keuangan rumah tangga Anda dengan lebih terstruktur dan mudah.

## 🌟 Fitur Utama
- **Sistem Login dengan PIN:** Menjaga privasi dan keamanan data keuangan keluarga Anda.
- **Pencatatan Transaksi:** Tambah pemasukan dan pengeluaran harian dengan cepat.
- **Manajemen Pos Keuangan:** Buat berbagai kategori (pos) pengeluaran seperti Kebutuhan Pokok, Pendidikan, Hiburan, Tabungan, dll.
- **Prioritas Pos:** Mengatur skala prioritas pada setiap pengeluaran.
- **Laporan & Visualisasi:** Melihat ringkasan pemasukan dan pengeluaran agar kondisi keuangan lebih mudah dipantau.
- **Tampilan Modern & Intuitif:** Desain UI/UX yang bersih dan nyaman digunakan (mendukung *material design*).

## 🛠️ Teknologi yang Digunakan
- **Platform:** Android (Minimum SDK disesuaikan)
- **Bahasa Pemrograman:** Java
- **UI System:** XML (Material Components)
- **Arsitektur:** Sesuai standar Android modern.

## 🚀 Cara Menjalankan Proyek (Lokal)
Jika Anda ingin menjalankan aplikasi ini di komputer Anda menggunakan Android Studio:

1. Pastikan Anda sudah menginstal [Android Studio](https://developer.android.com/studio).
2. Lakukan *Clone* repositori ini:
   ```bash
   git clone https://github.com/fauzanubaid-beep/UbaidFauzan_23101140087_06TPLE009_DompetKeluarga.git
   ```
3. Buka **Android Studio** dan pilih menu **Open**.
4. Cari dan pilih folder hasil *clone* tersebut.
5. Tunggu proses *Gradle Sync* selesai (pastikan komputer terhubung ke internet).
6. Klik tombol **Run** (segitiga hijau) atau tekan `Shift + F10` untuk menjalankan aplikasi pada Emulator atau Perangkat Android fisik Anda.
